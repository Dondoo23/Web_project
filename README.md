- 이슬, 황준수, 고경수



## 우리가 만드는 건 어떤 사이트인지?
- 스터디나 공모전 인원 모집 웹 사이트



## 구현해야할 기능
1)로그인 화면, 회원가입 화면
- 로그인
- 회원가입

2)main화면
- 게시판
  - 게시판에는 제목, 작성시간 띄우기
  - 올린 순서대로 게시판 띄우기
  - 검색 기능(제목, 내용)
  - 커서 갖다대면 색 변화주기
- 페이징
  - 현재 페이지에 변화주기(색 입히거나, 진하게)
- 글쓰기 버튼 

3)게시판 클릭 시
- 뒤로가기
- 내용 띄우기(제목, 내용, 글 작성시간)
- 댓글은 시간되면 나중에 구현

4)글쓰기 클릭 시
- 제목, 내용 입력 후 글쓰기 버튼 클릭 시 게시판에 띄우도록
